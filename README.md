## Features

| Feature | Description |
|---|---|
| ▶ Interview mode | Flashcard drill — random question, reveal answer, Got it / Missed it scoring, progress bar |
| ☾ Dark mode | One-click toggle in every guide and portal homepage. Saved across sessions |
| 🔍 Full-text search | Find any command, concept or error message instantly inside each guide |
| 🏷 Category filters | Filter questions by subtopic within each guide |
| ← Portal link | Every guide links back to the portal homepage |
| 📴 Offline-ready | Pure HTML — no build step, no Node.js, works without internet |
